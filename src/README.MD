Entidades Identificadas:

* model.Livro - título, autor e quantidade disponível 
* model.Aluno - ra, nome, telefone 
* Empréstimo - livro, aluno, data de empréstimo e data de devolução

Regras de negócio:

* Não deve ser possível emprestar um livro se não houver quantidade disponível.
* Não deve ser possível cadastrar livros com título vazio ou quantidade negativa.
